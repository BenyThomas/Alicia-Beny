Arrays Methods

Here we will discuss some basic array methods in JavaScript.
Slice method toString method Push method Pop method Shift method
Slice Method

The slice() method slices out a piece of an array and returns a new array containing the sliced out items.

The example below slices out a part of an array starting from array element 2 ("blue").

let colors = ["red", "yellow", "blue", "black", "white"]
let someColors = colors.slice(2)
  • This method creates a new array. It doesn't change the source array.
    • toString Method

    The JavaScript method toString(), converts an array to a string of comma separated array items.

    See the example below:

    let colors = ["red", "yellow", "blue", "black", "white"]
    let myColors = colors.toString();
  • The join() method works similarily, but a separator can be specified in addition.
    • Push Method

    The push() method adds a new element to an array, at the end.

    See the example below:

    let colors = ["red", "yellow", "blue", "black", "white"]
    colors.push("green") // adds a new element ("green") to colors
  • This method returns the new array length.
    • Pop Method

    The pop() method removes the last element from an array.

    See the example below:

    let colors = ["red", "yellow", "blue", "black", "white"]
    colors.pop()
  • This method returns the value that was "popped out"
